Secure IoT-Based Remote Solar Power Management for Grid Stability in Sri Lanka

Supervised by MM. Mohamed Mufassirin , South Eastern University of Sri Lanka

Reg. No: SEU/IS/19/ICT/054

This project develops a secure IoT system to monitor and control home-based solar power systems in real time, helping prevent grid instability in Sri Lanka. Using an ESP32 microcontroller and a SIM7600 4G module, the system collects voltage and current data via sensors and transmits it securely to the cloud using Firebase and MQTT with TLS encryption. A relay module allows remote control of solar inputs, while a ReactJS dashboard provides real-time monitoring for the Electricity Board. This solution addresses issues like the February 2025 blackout by enabling proactive grid management and ensuring stable power distribution with increasing solar adoption.
